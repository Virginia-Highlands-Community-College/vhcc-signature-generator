Step 3 - Add Your Signature and Update Font

Add your signature

  1. Copy your signature using the button in Step 2
  2. Open ⚙️ Settings → Account → Signatures
  3. Add a new signature, paste with Ctrl+V, and save
  4. Set defaults for new messages and replies/forwards

Set your font

  1. Go to Mail → Compose and reply
  2. Set default font to Arial and size to 11
  3. Click Save

Add your signature

  1. Copy your signature using the button in Step 2
  2. Go to File → Options → Mail
  3. Click Signatures... then New
  4. Name it and paste with Ctrl+V
  5. Set defaults for new and reply/forward messages
  6. Click OK

Set your font

  1. Go to File → Options → Mail
  2. Click Stationery and Fonts...
  3. Set fonts to Arial size 11 for new mail and replies/forwards, then save

  4. Set defaults for new mail and replies/forwards

Set your font

  1. Go to Outlook → Settings → Fonts
