Outlook Desktop (Windows/Mac)

  1. Copy your signature using the button above
  2. Open Outlook and go to File → Options → Mail
  3. Click Signatures...
  4. Click New to create a new signature
  5. Give it a name (e.g., "VHCC Signature")
  6. Click in the signature editor and press Ctrl+V (or Cmd+V on Mac) to paste
  7. Set as default for new messages and replies/forwards
  8. Click OK to save

Outlook on the Web

  1. Copy your signature using the button above
  2. Go to outlook.office.com
  3. Click the ⚙️ Settings gear icon (top right)
  4. Search for "Email signature" or go to Account → Signatures
  5. Click Add signature and give it a name (e.g., "VHCC Signature")
  6. In the text box, press Ctrl+V (or Cmd+V on Mac) to paste
  7. Enable Set default for new messages (and Set default for replies and forwards, if desired). Uncheck "Include a link to my bookings page" if you entered your Bookings URL in the generator.
  8. Click Save
