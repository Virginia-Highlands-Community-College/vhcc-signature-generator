Add Your Signature and Set Your Font

Add your signature

  1. Copy your signature using the button above
  2. Go to outlook.office.com
  3. Open ⚙️ Settings → Account → Signatures
  4. Add a new signature, paste with Ctrl+V, and save
  5. Set defaults for new messages and replies/forwards

Set your font

  1. Go to Mail → Compose and reply
  2. Set default font to Arial and size to 11
  3. Click Save

Add your signature

  1. Copy your signature using the button above
  2. Go to File → Options → Mail
  3. Click Signatures... then New
  4. Name it and paste with Ctrl+V
  5. Set defaults for new and reply/forward messages
  6. Click OK

Set your font

  1. Go to File → Options → Mail
  2. Click Stationery and Fonts...
  3. Set fonts to Arial size 11 for new mail and replies/forwards, then save

Add your signature

  1. Copy your signature using the button above
  2. In Outlook, open Outlook → Settings → Signatures
  3. Create a new signature and paste with Cmd+V
  4. Set defaults for new mail and replies/forwards

Set your font

  1. Go to Outlook → Settings → Fonts
  2. Set fonts for new messages, replies/forwards, and plain text
  3. Close settings to apply

  3. Paste or enter your mobile signature text

Add your signature

  1. In Outlook Android, open your profile icon → ⚙️ Settings
  2. Open Signature.
  3. Paste your signature from the signature generator

Set your font

  1. In ⚙️ Settings, go to Mail → Font
  2. Set Default Font to Arial and Default Size to 11
